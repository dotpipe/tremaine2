TreMaine ZIP 7 — Room Travel + Wrapped Dungeon

Features
- Dungeon walls remain visible
- Crossing room edges transitions rooms
- Pacman-style wrap between adjacent rooms
- World is room-based instead of one giant map
- Door tile (4) remains optional for explicit links

Rules
- Reach left edge -> enter right edge of neighboring room
- Reach right edge -> enter left edge of neighboring room
- Reach top edge -> enter bottom edge of neighboring room
- Reach bottom edge -> enter top edge of neighboring room
- Preserve player state
- Preserve room state
